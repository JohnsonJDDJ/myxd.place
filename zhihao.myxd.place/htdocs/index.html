Project: MineRift (2018 - 2019).

MineRift

Github:

Showcase Video:

This project is a Minecraft minimap that imitates the Summoner's Rift of the MOBA game League of Legends, developed by Riot Games. This project is developed during high school, during my class time (mostly) and free time (some). I revisited this map and was pretty amazed by the effort I put in back then. I think this project deserves some respect, so I made a showcase video and a Github repo to mark its existence in the history of the internet.

The whole idea is to play League of Legends in Minecraft (this project is literally created because I can't play league in high school but I can play minecraft). But, the game, as it develops, is now very different from League.

I strongly recommend checking our the showcase video. For more information and installation, checkout the github link.
