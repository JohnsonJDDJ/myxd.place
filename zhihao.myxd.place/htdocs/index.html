Riot Games API

This is an open source python package project. The package is called zilean, installable with pip. Please check out the links above.

Package zilean is designed to facilitate data analysis of the Riot MatchTimelineDto. The MatchTimelineDto is a powerful object that contains information of a specific League of Legends match at every minute mark. Naturally, the MatchTimelineDto became an ideal object for various machine learning tasks. For example, predicting match results using game statistics before the 16 minute mark.

The development of the package includes many tasks other than writing the source code. The tasks are: writing tests with coverage reports, setting up CI/CD pipelines through Github Actions, and constructing the documentation with Sphinx and Read the Docs.

The authors of the project are: Ingrid Chien, Johnson Du, Minori Jaggia, and Leif-Martin Saether Sunde.

This is a project attempting to predict forest fires in a reproducible manner. We conduct exploratory data analysis and use a random forest and logistic regression for prediction. The data is of two regions of Algeria, namely the Bejaia region located in the northeast of Algeria and the Sidi Bel-abbes region located in the northwest of Algeria. The time period is from June 2012 to September 2012.

Supporting the analysis is a robust reproduciblility framework. The tools and techniques include: Makefile commands to redo analysis and reproduce the figures, myBinder environment to run the analysis, GitHub Pages to present the analysis, and a utility python package for better code readability.

Please checkout the Github link for a complete set of information regarding the project.

Genshin Impact

This is a basic machine learning project that tries to predict the "damage calculation algorithm" in Genshin Impact, a game developed by miHoYo. I started playing the game shortly after the release and fell in love with the game quickly. As the game develops, I am intrigued with improving my performance (maximize the damage I deal to enemies) to the finest levels. Thus, I began this project to figure out how to maximize my damage with the equipmemts I currently have.

Through repeated in-game simulations, I collected data from the game and built a basic linear model to predict my damage using R. The result model is showcased through R ShinyApp (link above). I used this model and wrote some game tutorials that received 400,000+ views on the game's Chinese forum.

* Since I began this project purely out of my interest to the game, unfortunaley, I am unaware to perserve the code and the data at the beginning. The only code I currently have is the code for the ShinyApp.
